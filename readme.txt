== Description ==

Brightcove Portal will help you to add your brightcove videos to wordress from different brightcove accounts. Also you will be able to edit/delete videos from your brightcove account. 

For more information please visit [Plugin Info]http://svhosting.biz/wp-plugins/wp-brightcove-portal.html


== Installation ==

1. Extract the wp-brightcove-portal/ folder file to /wp-content/plugins/
1. Activate the plugin at your blog's Admin -> Plugins screen
1. The plugin will attempt to create a directory /wp-content/backup-*/ inside your WordPress directory.

== Frequently Asked Questions ==

=Why take some time to load all videos from brightcove=

It takes some time to load because the plugin gets the information from brightcove, then the plugin parse the information.
